# Develpment Process

```sh
# run install
npm install

# dev mode on develop on the browser
npm run dev

# build js
npm run build

# sync native plugin
npm run sync:android
npm run sync:ios

# copy build to native
npm run copy:android
npm run copy:ios

# run native
npm run android
npm run ios

# build native
npm run build:android
npm run build:ios
```
